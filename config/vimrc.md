# vimrc

文件放在在Linux用户目录下，在`$HOME/.vimrc`，全局路径在哪里可以问问搜索引擎

```
# 输入时搜索
set incsearch
# 显示命令
set showcmd
# 空格缩进
set shiftwidth=4
set expandtab
set tabstop=4
set softtabstop=4
# 按键映射
map <C-X> <Esc>:q!<CR>
imap <C-X> <Esc>:q!<CR>
map <C-R> <Esc>:e!<CR>
imap <C-R> <Esc>:e!<CR>
```
