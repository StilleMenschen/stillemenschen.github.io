# 云计算

## 五大基本特征

1. 按需获得自助服务
2. 广泛的网络接入
3. 资源池化
4. 快捷的弹性伸缩
5. 可计量的服务

## 四种服务

1. 云基础设施即服务（Infrastructure as a Service，IaaS）
2. 云平台即服务（Platform as a Service，PaaS）
3. 云软件即服务（Software as a Service，SaaS）
4. 云函数即服务（Functions as a Service，FaaS）

## 四类部署模式

1. 专（私）有云
2. 行业云
3. 公有云
4. 混合云

## 云架构

1. 应用平台层融合，敏捷化
2. 数据层融合
3. 基础资源设施融合
