# 预览 Git 仓库

## log

查看提交历史

```
git log [<options>] [<revision range>] [[--] <path>…​]
```

- 向下滚动
  - `j` 或 `Arrow Down`，滚动一行
  - `d` 滚动半页
  - `f` 或 `Page Down` 滚动一页
- 向上滚动
  - `k` 或 `Arrow Up`，滚动一行
  - `u` 滚动半页
  - `b` 或 `Page Up` 滚动一页
- 按 `q` 退出预览

### 选项

| 选项     | 说明                   |
| :------- | :--------------------- |
| --source | 每个提交展示具体的分支 |
